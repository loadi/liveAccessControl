# liveAccessControl
Пример ручной модерации доступа к сайту.

# Что делает?
Позволяет управлять каждым сеансом подключения к сайту.

# Как выглядит?
Админ панель реализована в виде таблицы, в которой клик по статусу доступа инвертирует его.
Таким образом возможно не только один раз разрешить доступ к веб-странице, но и запретить доступ в рамках одной сессии.
