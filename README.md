# liveAccessControl
Пример ручной модерации доступа к сайту.

# Что делает?
Позволяет управлять каждым сеансом подключения к сайту.

# Как выглядит?
Пользователь заполняет простую форму. 


![](https://sun9-66.userapi.com/impg/sB9zamzqSJMiwzuWc4pP5Ux6cURntl1GWUXLZQ/rasIPOKyKd4.jpg?size=300x45&quality=96&sign=d58ede6783bef8c1229cf53c7fff910f&type=album)


После ее отправки человек попадает в "комнату ожидания" до тех пор, пока вход не будет одобрен. Запрос на доступ передается администрации ресурса.


Панель администратора реализована в виде таблицы, в которой клик по статусу доступа инвертирует его.
Таким образом возможно не только один раз разрешить доступ к веб-странице, но и запретить доступ в рамках одной сессии.
